# Smart Parking System Project

## Team Work Division

### Member 1: Frontend (Folder: `client/`)
- **Role**: Solely responsible for the React application.
- **Tasks**:
  - Implement Landing Page, Dashboard, and Admin UI.
  - Design Glassmorphism styles in `client/src/App.css`.

### Member 2: Backend API (Folder: `server/member2_api/`)
- **Role**: Core API routing and server logic.
- **Tasks**:
  - Express server setup and REST endpoints.
  - Business logic for bookings.

### Member 3: Database & Auth (Folder: `server/member3_db/`)
- **Role**: SQL Schemas, Authentication, and Socket.io.
- **Tasks**:
  - PostgreSQL table definitions (Users, Slots, Bookings).
  - JWT integration and real-time event handlers.
  - Neon connection management.

### Member 4: IoT & Analytics (Folder: `server/member4_iot/`)
- **Role**: IoT Simulator and Admin backend logic.
- **Tasks**:
  - Hardware simulation script.
  - Revenue and occupancy data calculation.

## Tech Stack
- **Frontend**: Vite + React + Vanilla CSS (Glassmorphism design)
- **Backend**: Node.js + Express
- **Database**: Neon (PostgreSQL)
- **Real-time**: Socket.io
